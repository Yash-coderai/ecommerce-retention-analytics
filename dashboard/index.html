Project workflow

How the analysis is structured

Cleaning, balancing, training, evaluation, and visual reporting are separated clearly.

Data cleaning

Identifiers are removed, categorical values are standardized, and missing values are handled before any model training begins.

Feature engineering

Skewed variables are transformed, categories are one-hot encoded, and train/test features are aligned for consistent inference.

Model evaluation

Random Forest and LightGBM are compared using accuracy, precision, recall, F1, and ROC-AUC with matching visual outputs.
